js基础系列（二）数据类型

发表于 2020-07-30 更新于 2020-08-05

js数据类型有七种，分为基本类型和引用类型

基本类型：string、boolean、number、undefined、null、symbol
引用类型（也叫复杂类型）：Object
引用类型中又可分为：狭义的object、function、array、Date、RegExp等

1、数据类型检测手段

由于js是松散的，所以需要方法来判断给定值的数据类型，目前有三种方法

1）typeof

typeof '123' // "string"
typeof 123 // "number"
typeof false // "boolean"
typeof undefined // "undefined"
typeof Symbol() // "symbol"
typeof null // "object"
typeof { } // "object"
typeof [] // "object"
typeof new Date // "object"
typeof new RegExp // "object"
typeof function() {} // "function"

typeof只能判断除了null之外的基本数据类型，那如果我想区分{}和[]该怎么判断？可以用Object.prototype.toString

2）Object.prototype.toString

Object.prototype.toString.call(1) // "[object Number]"
Object.prototype.toString.call('hi') // "[object String]"
Object.prototype.toString.call({a:'hi'}) // "[object Object]"
Object.prototype.toString.call([1,'a']) // "[object Array]"
Object.prototype.toString.call(true) // "[object Boolean]"
Object.prototype.toString.call(() => {}) // "[object Function]"
Object.prototype.toString.call(null) // "[object Null]"
Object.prototype.toString.call(undefined) // "[object Undefined]"
Object.prototype.toString.call(Symbol(1)) // "[object Symbol]"
Object.prototype.toString.call(Math) // "[object Math]"
Object.prototype.toString.call(new Date) // "[object Date]"
Object.prototype.toString.call(Math) // "[object Math]"

利用Object.prototype.toString可以做出比typeof更加精准的判断

2）instanceof

  • instanceof表示对象是否为某个构造函数的实例，返回的是布尔值
  • instanceof只能用于对象，不适用于基本数据类型

2、string类型

  • 反斜杠用来表示转义，比如常用的有换行、回车等，用到了再去查。

  • 把一个值转换为字符串有两个方法。第一个是toString()，数值、布尔值、对象和字符串值都有 toString()方法，但 null 和 undefined 值没有这个方法。

    第二个是函数String()，这个函数能够将任何类型的值转换为字符串。如果值是null，则返回’null’。如果值是undefined，则返回’undefined’。

  • 在ES6中，新增了字符串模版

  • 实例属性

Number.prototype.toLocaleString() // 接受一个地区码作为参数，返回一个字符串，表示当前数字在该地区的当地书写形式。

    5、undefined和null类型

1）undefined

  • undefined表示未定义，当var定义了没有赋值，这是js默认值为undefined
  • 之前在用antd的select组件时，当初始值为空字符串时，placeholder属性就不起作用，初始值设为undefined就可以了。

2）null

  • null表示的是空值，空指针对象。所以 typeof null 为object
  • 如果定义的变量准备在将来用于保存对象，最好将其初始化为null

6、symbol类型

symbol表示独一无二的值，从根本上防止属性名的冲突。其中的细节请参考阮一峰老师的ES6入门

3、let和const与var的区别

let和const是在ES6中提出的，和var最大的区别是let和const会产生块级作用域。在js中有全局作用域和函数作用域（eval几乎不用，就不考虑了），没有块级作用域。let和const实际上为js增加了块级作用域。更多例子和细节请参考阮一峰老师的ECMAScript6入门
